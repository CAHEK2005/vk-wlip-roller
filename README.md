# VK Cloud Whitelist IP Roller

Веб-интерфейс для поиска и замены Floating IP в VK Cloud — крутит адреса до тех пор, пока не найдёт IP из белого списка российских мобильных подсетей.

## Возможности

- Поиск нового Floating IP из белого списка (~30 000 CIDR-подсетей)
- Замена существующего IP: удаляет старый, подбирает новый из белого списка
- Несколько аккаунтов VK Cloud — параллельный поиск
- Прокси (SOCKS5/HTTP) для каждого аккаунта отдельно
- Таймер — автоматическая остановка через заданное время
- Персистентные сессии — при закрытии вкладки поиск продолжается в фоне; при возврате показывается весь накопленный лог
- Антифрод: случайные паузы, длинные паузы с заданным шансом, User-Agent ротация, превентивное обновление токена

## Как работает

1. Выделяет новый Floating IP через OpenStack Neutron API
2. Проверяет попадание в одну из ~30 000 CIDR-подсетей из белого списка
3. Если нет — удаляет IP и пробует снова
4. Если да — сообщает об успехе и останавливается

Прогресс отображается в реальном времени через SSE-поток.

## Установка на Ubuntu

Одной командой (требует root):

```bash
curl -fsSL https://raw.githubusercontent.com/CAHEK2005/vk-wlip-roller/main/install.sh | sudo bash
```

Скрипт:
- устанавливает зависимости (Python 3, git)
- клонирует репозиторий в `/opt/vk-wlip-roller`
- создаёт Python venv и устанавливает зависимости
- регистрирует и запускает systemd-сервис
- выводит IP-адрес и случайный порт, по которому доступен интерфейс

## Обновление

```bash
curl -fsSL https://raw.githubusercontent.com/CAHEK2005/vk-wlip-roller/main/update.sh | sudo bash
```

или если уже установлен:

```bash
sudo bash /opt/vk-wlip-roller/update.sh
```

## Ручной запуск (для разработки)

```bash
git clone https://github.com/CAHEK2005/vk-wlip-roller
cd vk-wlip-roller
pip install -r requirements.txt
python app.py
```

Откройте в браузере: [http://localhost:8000](http://localhost:8000)

## Настройка

### 1. OpenRC v3 файл

Скачайте на сайте VK Cloud:
**Проект → Настройки → Доступ по API → Скачать openrc v3**

### 2. API Токен

Создайте постоянный токен:
**Аккаунт → Безопасность → API-ключи → Создать ключ**

Оба файла указаны в `.gitignore` и не попадут в репозиторий.

## Использование

### Найти новый IP из белого списка

1. Добавьте аккаунт, загрузите OpenRC файл, введите API токен
2. При необходимости укажите прокси и нажмите «Проверить»
3. Настройте интервал паузы и таймер (опционально)
4. Нажмите **Запустить поиск на выбранных**

### Заменить существующий IP

1. Нажмите **Загрузить IP** для нужного аккаунта
2. Кликните на IP в списке, затем нажмите **Заменить выбранный IP**

Старый IP удаляется только после того, как найден новый из белого списка.

### Сессии в фоне

Поиск продолжается даже при закрытии вкладки браузера. При следующем открытии страницы появится баннер «Активная сессия найдена» — нажмите **Подключиться** для просмотра прогресса.

## Белый список

Используется: [russia-mobile-internet-whitelist](https://github.com/hxehex/russia-mobile-internet-whitelist/blob/main/cidrwhitelist.txt)
Файл `cidrwhitelist.txt` загружается при старте сервера (~30 000 сетей).
